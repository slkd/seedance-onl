🚀 Getting Started with Seedance 2.0

Step 1: Access Dreamina Platform

Seedance 2.0 is available through ByteDance's Dreamina platform. Create a free account to get 50 trial credits.

Step 2: Select Seedance 2.0

From the Dreamina dashboard, choose Seedance 2.0 from the AI model selector. You'll see options for different versions (1.0, 1.5, 2.0).

Step 3: Understand Credits

Video generation costs credits based on:

  • Duration (5s = 10 credits, 15s = 50 credits)
  • Resolution (1080p vs 2K)
  • Complexity (multi-modal inputs cost more)

📹 Basic Video Generation (Text-Only)

Simple Prompt Example

Prompt: "A serene beach at sunset with waves gently crashing on shore. 
Cinematic camera slowly panning left. Golden hour lighting."

Settings:
- Duration: 10 seconds
- Resolution: 2K
- Style: Photorealistic

Best Practices for Text Prompts

  • Be specific - "Red sports car" vs "car"
  • Include camera movement - "slow zoom in", "tracking shot"
  • Describe lighting - "golden hour", "dramatic shadows"
  • Set the style - "cinematic", "anime", "realistic"

🎨 Multi-Modal Video Generation

This is where Seedance 2.0 shines. Combine multiple input types for precise control.

Example: Character-Consistent Video

Workflow:

  1. Upload character image - PNG of your desired character
  2. Upload motion reference - Short video showing desired movement
  3. Upload audio - Dialogue or music track
  4. Write prompt with @ mentions:
Prompt: "@character is walking through a busy city street following 
the movement style from @motion_reference. Synchronize lip movement 
with @dialogue_audio. Cinematic shot with shallow depth of field."

Understanding @ Mentions

The @ system lets you explicitly reference uploaded assets:

  • @character_image - Use this face/design
  • @motion_video - Copy this movement
  • @camera_ref - Replicate this camera work
  • @audio_track - Sync to this audio

✍️ Seedance Prompt Engineering Guide

Prompt Structure Template

[SUBJECT] + [ACTION] + [ENVIRONMENT] + [CAMERA] + [STYLE] + [LIGHTING]

Example:
"A young woman (@character) dancing gracefully (@dance_reference) 
in a futuristic neon city (cyberpunk environment) 
with a sweeping crane shot (@camera_movement) 
in a cinematic style with vibrant neon lighting"

Advanced Prompting Tips

Goal Technique Example
Better consistency Use character references "Same @character from earlier scene"
Precise motion Reference video clips "Follow the choreography in @dance_clip"
Matching aesthetics Style transfer "Apply the visual style from @reference_image"

💡 Pro Tips & Best Practices

Tip #1: Start Simple, Then Iterate

Generate a basic version first, then use the "extend" feature to add complexity.

Tip #2: Use High-Quality References

Better input assets = better output. Use clear, well-lit reference images.

Tip #3: Optimize Credit Usage

  • Test with 5-second generations before going longer
  • Use 1080p for drafts, 2K only for finals
  • Save successful prompts for reuse

Tip #4: Master the @ Mention System

This is Seedance's secret weapon. Practice referencing specific elements.

Tip #5: Join the Community

Learn from others on Seedance Reddit and Discord.

🔧 Common Issues & Solutions

Video doesn't match my prompt
Solution: Be more specific in your prompt. Add camera instructions, lighting details, and use @ mentions for precise references.
Character consistency issues
Solution: Upload a clear character reference image and explicitly mention it with @character in every prompt.
Generation is too slow
Solution: Upgrade to Creator plan for priority queue, or generate during off-peak hours.
