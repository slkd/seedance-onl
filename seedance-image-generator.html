🎨 Seedance Image Generation

While Seedance is primarily known for AI video, the Dreamina platform includes powerful image generation tools that work seamlessly with Seedance video workflows.

🔄 Image → Video Workflow

  1. Generate character/scene in Image Generator
  2. Save to Dreamina library
  3. Reference in Seedance with @ mentions
  4. Create consistent videos using your generated images

💡 Use Cases

  • Character design: Create consistent character references
  • Environment concepts: Generate scene backgrounds
  • Style references: Establish visual aesthetics
  • Storyboarding: Plan video sequences
